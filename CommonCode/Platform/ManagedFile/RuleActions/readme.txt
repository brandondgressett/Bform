You might be wondering why ManagedFile has no rule actions.

There just aren't any use cases so far where a rule attaches a file 
to something instead of a user doing it. You might make the case
that deleting files is a useful rule action, but we still haven't
seen a real need for it yet.

Until we think of a need, this is just considered "too fancy."

If you're reading this, you might have a use case for it. 
Well ... we'll get to it eventually, when a use case we need requires it. 
Or maybe you could help us.
